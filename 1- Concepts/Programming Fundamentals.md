
### `Definition`

Programming fundamentals are the essential building blocks shared across all programming languages. They describe the core structures, behaviors, and concepts required to write, understand, and reason about any program.

---
### `Key Ideas`

- Provide the foundation for all languages and paradigms.
    
- Include basic structures like data, logic, control flow, and functions.
    
- Explain how code interacts with memory and executes step-by-step.
    
- Act as the parent category for core subtopics like logic, types, and variables.
