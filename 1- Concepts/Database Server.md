
### `Definition`

A **database server** is a system component responsible for storing, managing, and providing access to persistent data through a database management system (DBMS).

---
### `Key Ideas`

- **Persistent Storage:**  
    Stores application data reliably and durably.
    
- **Controlled Data Access:**  
    Data is accessed via queries issued by applications or services.
    
- **Managed by a DBMS:**  
    Software that enforces consistency, security, and integrity.
    
- **Performance-Critical Component:**  
    Often a bottleneck if poorly designed or scaled.
    
- **High Availability Considerations:**  
    Replication, backups, and failover are common in production.
