
### `Definition`

Functions are reusable blocks of code that perform a specific operation and can be executed multiple times. They take inputs, process them, and optionally return a value.

---
### `Key Ideas`

- Encapsulate a specific behavior or computation.
    
- Can accept **parameters** and return **outputs**.
    
- Improve readability, modularity, and reuse.
    
- Execute only when called/invoked.
    
- Independent of paradigms â€” used in procedural, OOP, and functional styles.
